# KPIs Hospitalarios — Estándar MINSA/EsSalud

## Indicadores de Producción
| KPI | Fórmula | Meta MINSA |
|-----|---------|-----------|
| Consultas/médico/día | Consultas totales / médicos activos / días | 12-16 |
| Rendimiento cama | Egresos / camas disponibles | > 40/año |
| Promedio estadía | Días estancia / egresos | < 6 días |
| Giro cama | Egresos / camas / período | > 3.5 por mes |

## Indicadores de Calidad
| KPI | Meta |
|-----|------|
| SLA emergencia crítica | < 15 min primer contacto médico |
| SLA emergencia urgente | < 30 min |
| Mortalidad intrahospitalaria | < 25 × 1,000 egresos |
| Tasa reingreso 30 días | < 8% |

## Indicadores de Acceso
| KPI | Meta |
|-----|------|
| Tasa de ocupación camas | 85-90% |
| Lista espera especialidad | < 30 días |
| IRAB (Índice Rechazo por Baja Cap.) | < 2% |
